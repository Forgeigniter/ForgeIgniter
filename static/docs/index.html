Introduction

You should be able to find every scrap of information needed to use Halogy and its Modules, be reminded that this version of Halogy is the Community Edition, an extension of the default 1.2 core release. This version includes all the latest updates, bug fixes and feature updates from the Community, this often includes things like framework updates and new features to the modules.

About The Documentation

A lot of time has been spent putting the documentation together to make things easy to follow. Halogy on it's own is very intuitive and has been put together in such a way that almost anyone could easily pick things up but on that odd occasion you might want a more detailed guide or want to know a full tag reference for a specific module, this is where the documentation plays its part well.

What's happened ?

In the change log you can find every thing that's happened, here you'll find all the updates/fixs and enhancements to Halogy CE this has been moved from the text document and place here instead for a better viewing experience.

Some people thought Halogy was a dead project as nothing had been released for a long time but be rest assured that its very far from that and many of us are working hard on improvements and new features to Halogy. Halogy 2 is still being worked on and should be with us soon, if you're specifically wondering about a date, I guess it'll be done when it's done, mean while you can enjoy the CE version.
